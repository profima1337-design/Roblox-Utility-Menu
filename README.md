# Roblox Utility Menu

## 📄 Описание
**Roblox Utility Menu** — лёгкое Lua‑меню для инджекторов (Xeno и аналогичных).  
Скрипт выполнен в формате одного файла и предоставляет удобную панель управления с набором клиентских функций для Roblox.  
Меню не требует дополнительных модулей и запускается напрямую через инджектор.

---

## 🚀 Как использовать
1. Открыть инджектор.  
2. Выполнить инджект в Roblox.  
3. Скачать файл `menu.lua` и загрузить его в инджектор.  
4. Запустить скрипт.

---

## ✨ Основные функции
- **ESP** — подсветка игроков и объектов  
- **Aimbot** — помощь в наведении  
- **Infinity Jump** — бесконечные прыжки  
- **Checkpoint** — сохранение позиции  
- **Fly** — свободный полёт  
- **Noclip** — проход сквозь стены  
- **Speed** — изменение скорости передвижения  
- и другие вспомогательные функции
